About This Project

PixelVista is my frontend development project created for the CodeAlpha Internship. It demonstrates responsive web design, JavaScript DOM manipulation, category filtering, and an interactive lightbox with smooth animations. It includes category filters, lightbox preview, previous-next navigation, smooth animations, responsive layout, and keyboard navigation.
